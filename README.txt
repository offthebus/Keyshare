Keyshare. Console mode wow key sharer.
Simplest broadcaster for WoW boxing I could think of.
Works on 1 machine. Tested with 5mans. No config. 
First wow window detected is master. Hard coded window layout.
Hard coded list of keys to broadcast.
Hard coded mouse to arrow key mappings.
To install: download keyshare.exe and put it anywhere.
To use: 
(1) ensure no Wow windows are open 
(2) run keyshare.exe
(3) open master session
(4) open other sessions
(5) enter l(ayout) command
That's it.
Type h(elp) for info.
